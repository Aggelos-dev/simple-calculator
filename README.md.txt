# Simple Calculator

Αυτή είναι μια απλή αριθμομηχανή γραμμένη σε **HTML, CSS και JavaScript**.

## Λειτουργίες

* Προσθέσεις (`+`)
* Αφαιρέσεις (`-`)
* Πολλαπλασιασμός (`×`)
* Διαίρεση (`÷`)
* Backspace (`←`)
* Clear (`C`)
* Υπολογισμός με `=`

## Χρήση

1. Άνοιξε το `index.html` σε οποιονδήποτε browser.
2. Πάτα τα κουμπιά για να εισάγεις αριθμούς και πράξεις.
3. Δες το αποτέλεσμα στην οθόνη.

## Δομή Project

```
calculator/
├─ index.html
├─ style.css
├─ script.js
└─ README.md
```

## Σημειώσεις

* Υποστηρίζονται μόνο αριθμοί και βασικές πράξεις.
* Οι αριθμοί εμφανίζονται στην οθόνη ως string.
* Δεν υπάρχει υποστήριξη για ιστορικό πράξεων ή δεκαδικούς αν δεν προσθέσεις κουμπί `.`.
